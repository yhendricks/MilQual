About Test Configurations

Test configurations define what measurements and questions should be collected when testing PCBs of a specific type. Each PCB type can have multiple test configurations to accommodate different testing requirements.

Each test configuration consists of:

  • Parameters: Quantitative measurements like voltage, current, frequency, resistance, temperature, etc.
  • Questions: Qualitative yes/no questions that must be answered during testing
